# Monica's 24-7 Bail Bonds

Professional website for Monica's 24-7 Bail Bonds in Kissimmee, FL.

## Features
- **Responsive Design**: Works perfectly on mobile and desktop.
- **Theme Toggle**: Switch between Professional Navy (Dark) and Clean White (Light) modes.
- **Fast & Static**: Built with HTML5, CSS3, and Vanilla JavaScript for optimal speed.
- **SEO Optimized**: Includes proper meta tags and semantic structure.

## Deployment
This site is designed to be hosted on **GitHub Pages**.
